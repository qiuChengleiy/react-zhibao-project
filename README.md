# 质保后台


#### 项目结构 

``` sh
├── mock/ # 网络请求数据模拟
├── node_modules/ # 包模块文件
├── public/ # 公共目录
	├── index.html  # 首页模板
├── src/ # 开发目录
    ├── api/ # api管理
    ├── assets/ # 静态资源
    ├── components/ # 组件
    ├── config/ # 相关配置
    ├── models/ # 状态管理
    ├── pages/ #  页面
    ├── services/ # API
    ├── utils/ # 工具类
    ├── index.css #  公共样式
    ├── index.js #  应用入口
    ├── router.js # 路由管理

```
